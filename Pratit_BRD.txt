Business Requirement Document (BRD)
Project Title: Student Exam Performance Analysis Dashboard
1. Project Overview
The objective of this dashboard is to analyze student academic performance using different educational, behavioral, and demographic factors. The dashboard helps users identify trends related to attendance, study habits, internet access, tutoring, GPA, and performance levels.
The dashboard is developed in Power BI with interactive slicers, KPI cards, and analytical charts for better decision-making.
2. Dashboard Objectives
Monitor student academic performance
Analyze factors affecting exam scores
Compare student behavior and study patterns
Identify pass/fail distribution
Evaluate impact of tutoring, attendance, and internet access
Provide interactive filtering for detailed analysis
3. Data Cleaning Process
Removed duplicate records
Checked null values and replaced missing data where required
Corrected inconsistent categorical values
Verified numerical columns datatype
Removed unnecessary columns
Used Describe() analysis for statistical understanding
Standardized category names
4. Dashboard Structure
The dashboard contains 2 Pages:
Academic Performance Overview
Student Lifestyle & Support Analysis
PAGE 1 – Academic Performance Overview
Page Name:
Academic Performance Overview Dashboard
KPI Cards
1. Average Assignment Completion
Formula:
Average Assignment Completion = AVERAGE(assignment_completion_rate)
2. Average Attendance
Formula:
Average Attendance = AVERAGE(attendance_rate)
3. Average Final Score
Formula:
Average Final Score = AVERAGE(final_exam_score)
4. Average GPA
Formula:
Average GPA = AVERAGE(previous_gpa)
